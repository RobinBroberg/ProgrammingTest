Programming Test

Build the project using Gradle:

./gradlew build

Applications
The project includes the following applications:

Anagram Checker
An application to check if two words are anagrams.
Run the application with the following command:

./gradlew runAnagramChecker

Link Finder
An application to find and extract links from a web page.
Run the application with the following command:

./gradlew runLinkFinder

Calculator One
An application to perform calculations with two numbers and one operator.
Run the application with the following command:

./gradlew runCalculatorOne

Calculator Two
An application to perform calculations with operators of the same precedence.
Run the application with the following command:

./gradlew runCalculatorTwo

Calculator Three
An application to perform calculations with any operator and any amount of numbers.
Run the application with the following command:

./gradlew runCalculatorThree

Run tests with the following command:

./gradlew test
